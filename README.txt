CSCE 3444 Section 001 Restaurant Software Project

ALL FILES NEEDED TO RUN APP INSIDE KKWH_APP FOLDER
----LOCALLY HOSTING DATABASE INSTRUCTIONS----

    download xampp
    launch xampp in port 8080
    in xampp, start apache and mySQL
    in browser, enter domain localhost/phpmyadmin
    create new database called "restaurantapp" without the quotes
    in database, click SQL tab.
    copy queries inside file buildTables.sql into the text box, then click go
    copy MyApi directory into C:\xampp\htdocs

----RUNNING APP (ANDROID STUDIO EMULATOR)----

    open android studio
    import 3444-k-kamp-develop folder
    click run project(pressing green play button at the top)
    accept any prompts that may pop up
